# üìã Simple Copy-Paste Guide (5 Steps Per File)

## The Process for EVERY File

### In Figma Make (this window):

**Step 1:** Look at the LEFT side of this window
- You'll see a list of files (App.tsx, main.tsx, etc.)

**Step 2:** Click on a file name
- The code appears in the main area

**Step 3:** Select all the code
- Click in the code area
- Press **Ctrl+A** (Windows) or **Cmd+A** (Mac)
- Everything turns blue/highlighted

**Step 4:** Copy the code  
- Press **Ctrl+C** (Windows) or **Cmd+C** (Mac)

### On Your Computer:

**Step 5:** Create a new file and paste
- Open Notepad or VS Code
- Paste with **Ctrl+V** (Windows) or **Cmd+V** (Mac)
- Save with the SAME filename (e.g., `App.tsx`)
